# AyurSutra Chatbot – AI Based Ayurvedic Health Guide

## Overview
AyurSutra is an AI powered chatbot that detects a user's Prakruti (Vata, Pitta, Kapha) and provides personalized Ayurvedic health recommendations.

## Features
- Prakruti prediction using Machine Learning
- Ayurvedic health recommendations
- Chatbot interface
- Web based UI

## Technologies Used
- Python
- Machine Learning
- NLP
- Streamlit
- Pandas
- Scikit-learn

## Dataset
Contains Ayurvedic symptoms and remedies dataset.
